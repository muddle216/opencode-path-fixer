Privacy Policy

OpenCode Path Fixer

Last updated: 2026-05-11

Data Collection

This extension does not collect, transmit, or store any personal information. All processing is performed locally within your browser.

localStorage Usage

This extension reads and modifies path data in your browser's localStorage to fix Windows path separator issues. This data remains entirely on your device and is never sent to any external server.

External Communications

This extension makes no network requests. It operates entirely offline on locally stored data.

Permissions

This extension requests minimal permissions required only to function on web pages you visit. It does not access unrelated data.

Changes to This Policy

If this privacy policy changes, the updated version will be posted at this URL.

Contact

For questions about this privacy policy, please open an issue on the GitHub repository.
